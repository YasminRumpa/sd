1. Discuss the scope of var let const.
Ans: var- var is a functional scope. var can be accessed from anywhere in the code.
let- let is a block scope element. let can be changed.
const- const is also a block scope element. it can't be changed.

2. The use cases of null and undefined.
Ans: Null means when a variable has no value and undefined means variable has been declared but no value has assigned.

3. What do you mean by RestAPI?
Ans: A set of rules that says how applications can connect to and communicate with each other.
